# [SSU](https://ssu.oom-wg.dev)

SSU是一款闭源软件，但是其分发仍使用[F2DLPRL](https://github.com/OOM-WG/F2DLPRL)，使用者需要遵守其相关规定。

[FVV](https://github.com/OOM-WG/FVV)、[NGA SDK](https://github.com/TianwanTW/NGA-SDK)以及[CppLess](https://github.com/OOM-WG/CppLess)是SSU的三个第一方开源依赖，如有兴趣可以前去查看。

官方开发成员无力进行多语言维护，仅能维护简中并机翻部分语言，如想为SSU的多语言事业出一份力，可为此仓库发出拉取请求。
